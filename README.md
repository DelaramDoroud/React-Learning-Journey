# React Practice Projects 🚀

This repository contains a collection of React practice projects completed during my React learning journey. Each project focuses on a specific concept, helping me build a solid understanding of React fundamentals and intermediate topics.

---

## 📚 Projects Overview

### 1. Simple Calculator
A basic calculator built with React using `useState` and `useRef`.  
Supports addition, subtraction, multiplication, and division operations.

### 2. Counter App
A simple counter application demonstrating state management with `useState` and event handling.

### 3. User Fetch App
Fetches random user data from an external API using `useEffect` and displays it with conditional rendering.

### 4. Fruits List & Counter (State Sharing)
Demonstrates how to share state between components using props and illustrates the concept of lifting state up.

### 5. Desserts List (Filter & Sort)
Filters and sorts a list of desserts using JavaScript array methods (`filter`, `sort`, `map`) before rendering.

### 6. Signup Form with Validation
A controlled form with validation logic, error handling, and dynamic UI feedback.

### 7. Theme Switcher (Context API)
Implements global state management using React Context API and a custom hook to toggle between light and dark mode.

### 8. Gift Card App
Manages complex state using objects and demonstrates immutable updates and conditional rendering.

## 🧠 Concepts Covered

- React Hooks (`useState`, `useEffect`, `useRef`, `useContext`)

- API integration
- Context API (global state management)
- Data transformation (`filter`, `sort`, `map`)
